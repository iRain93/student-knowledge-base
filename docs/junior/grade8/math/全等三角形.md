# 全等三角形

## 学习目标

1. 理解全等三角形的概念
2. 掌握全等三角形的判定方法
3. 能进行全等三角形的证明

## 全等三角形的概念

能够完全重合的两个三角形叫做全等三角形。全等用符号"≌"表示。

全等三角形的对应边相等，对应角相等。

## 判定方法

| 方法 | 条件 | 说明 |
|---|------|------|
| SSS | 三边对应相等 | 边边边 |
| SAS | 两边及其夹角对应相等 | 边角边 |
| ASA | 两角及其夹边对应相等 | 角边角 |
| AAS | 两角及其中一角的对边对应相等 | 角角边 |
| HL | 斜边和一条直角边对应相等 | 仅适用于直角三角形 |

## 证明的基本格式

```
在△ABC 和△DEF 中，
  AB = DE（已知）
  ∠A = ∠D（已知）
  AC = DF（已知）
∴ △ABC ≌ △DEF（SAS）
```
