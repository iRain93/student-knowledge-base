# 三角形

## 学习目标

1. 理解三角形的基本概念和分类
2. 掌握三角形内角和定理
3. 掌握三角形三边关系
4. 理解三角形的高、中线、角平分线

## 三角形的分类

**按角分**：
| 类型 | 特征 |
|------|------|
| 锐角三角形 | 三个角都是锐角 |
| 直角三角形 | 有一个角是直角（90°） |
| 钝角三角形 | 有一个角是钝角 |

**按边分**：
| 类型 | 特征 |
|------|------|
| 不等边三角形 | 三边都不相等 |
| 等腰三角形 | 有两条边相等 |
| 等边三角形 | 三条边都相等 |

## 三角形内角和定理

**三角形三个内角的和等于 180°。**

∠A + ∠B + ∠C = 180°

## 三角形三边关系

**三角形任意两边之和大于第三边。**

a + b > c，a + c > b，b + c > a

## 三角形的主要线段

| 线段 | 定义 |
|------|------|
| 高 | 从一个顶点向对边作垂线 |
| 中线 | 连接顶点与对边中点 |
| 角平分线 | 平分一个角，交对边于一点 |
