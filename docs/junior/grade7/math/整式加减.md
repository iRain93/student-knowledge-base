# 整式加减

## 学习目标

1. 理解单项式、多项式、整式的概念
2. 掌握合并同类项
3. 掌握去括号法则
4. 会进行整式的加减运算

## 基本概念

| 概念 | 定义 | 示例 |
|------|------|------|
| 单项式 | 数字与字母的乘积 | 3x², -5ab, 2 |
| 多项式 | 几个单项式的和 | 3x² - 2x + 1 |
| 整式 | 单项式和多项式的统称 | — |
| 同类项 | 字母相同且指数相同的项 | 3x² 和 -5x² |

## 合并同类项

**系数相加，字母和指数不变。**

::: tip 示例
3x² + 5x² - 2x² = (3 + 5 - 2)x² = 6x²
:::

## 去括号法则

- 括号前是"+"：去掉括号，各项不变号
- 括号前是"-"：去掉括号，各项都变号

::: tip 示例
a + (b - c) = a + b - c

a - (b - c) = a - b + c
:::

## 整式加减步骤

1. 去括号
2. 找同类项
3. 合并同类项
