# 相似三角形

## 学习目标

1. 理解相似三角形的概念
2. 掌握相似三角形的判定方法
3. 理解相似比和相似三角形的性质
4. 了解位似图形

## 相似三角形的概念

对应角相等、对应边成比例的两个三角形叫做相似三角形。

相似比（相似系数）：对应边的比值 k。

## 判定方法

| 方法 | 条件 |
|---|------|
| AA | 两角对应相等 |
| SAS | 两边对应成比例且夹角相等 |
| SSS | 三边对应成比例 |

## 相似三角形的性质

1. 对应角相等
2. 对应边成比例
3. 周长比 = 相似比
4. 面积比 = 相似比的平方

## 相似三角形的高、中线、角平分线

相似三角形对应高、对应中线、对应角平分线的比都等于相似比。

## 位似

两个图形不仅是相似图形，而且对应顶点的连线相交于一点，对应边互相平行，这样的两个图形叫做位似图形。
