# 二次函数

## 学习目标

1. 理解二次函数的概念
2. 掌握二次函数的图像和性质
3. 会求二次函数的顶点、对称轴、最值
4. 能用二次函数解决实际问题

## 二次函数的概念

一般地，形如 **y = ax² + bx + c（a ≠ 0）** 的函数叫做二次函数。

## 图像与性质

**二次函数的图像是一条抛物线。**

| a 的值 | 开口方向 | 最值 |
|---|---|------|
| a > 0 | 向上 | 有最小值 |
| a < 0 | 向下 | 有最大值 |

## 三种形式

| 形式 | 表达式 | 用途 |
|------|--------|------|
| 一般式 | y = ax² + bx + c | 求与 y 轴交点 |
| 顶点式 | y = a(x - h)² + k | 顶点坐标 (h, k) |
| 交点式 | y = a(x - x₁)(x - x₂) | 与 x 轴交点 |

## 关键公式

- **对称轴**：x = -b / (2a)
- **顶点坐标**：(-b/(2a), (4ac - b²)/(4a))

## 二次函数与一元二次方程

二次函数 y = ax² + bx + c 与 x 轴的交点个数：

| 判别式 Δ = b² - 4ac | 交点个数 |
|---|---|
| Δ > 0 | 2 个 |
| Δ = 0 | 1 个 |
| Δ < 0 | 0 个 |
