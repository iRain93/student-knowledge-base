# 圆

## 学习目标

1. 理解圆的有关概念
2. 掌握垂径定理和圆周角定理
3. 掌握直线与圆、圆与圆的位置关系
4. 掌握弧长和扇形面积公式

## 基本概念

| 概念 | 定义 |
|------|------|
| 圆心 | 圆的中心点 O |
| 半径 | 圆心到圆上任意一点的线段 |
| 直径 | 经过圆心的弦，是半径的 2 倍 |
| 弦 | 连接圆上任意两点的线段 |
| 弧 | 圆上任意两点间的部分 |

## 垂径定理

**垂直于弦的直径平分这条弦，并且平分弦所对的两条弧。**

## 圆周角定理

**同弧所对的圆周角相等，且等于圆心角的一半。**

## 直线与圆的位置关系

| 位置关系 | 条件（d 为圆心到直线距离） |
|---|------|
| 相交 | d < r（2 个交点） |
| 相切 | d = r（1 个交点） |
| 相离 | d > r（0 个交点） |

## 弧长与扇形面积

- **弧长**：l = nπr / 180
- **扇形面积**：S = nπr² / 360 = (1/2)lr
