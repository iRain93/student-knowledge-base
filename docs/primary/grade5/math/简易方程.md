# 简易方程

## 学习目标

1. 理解用字母表示数的意义
2. 理解方程的概念
3. 会解简易方程

## 用字母表示数

- 加法交换律：a + b = b + a
- 乘法分配律：a × (b + c) = a × b + a × c
- 正方形的周长：C = 4a

## 方程的概念

**含有未知数的等式叫做方程。**

::: tip 辨析
- 3 + 5 = 8 → 是等式，但不是方程（没有未知数）
- x + 5 → 不是等式，不是方程
- x + 5 = 8 → 是方程（有未知数，是等式）
:::

## 解方程

**利用等式的性质**：等式两边同时加、减、乘、除以同一个非零数，等式仍然成立。

::: tip 示例：解 x + 3 = 8
x + 3 - 3 = 8 - 3

x = 5
:::

## 常见问题

**Q：孩子不理解为什么要用字母表示数？**
A：用字母表示数是为了表达一般规律，是代数思维的起点。可以举生活中的例子来理解。
