# 分数加减法

## 学习目标

1. 掌握同分母分数加减法
2. 掌握异分母分数加减法（通分）
3. 理解约分和最简分数

## 同分母分数加减

**分母不变，分子相加减。**

$$\frac{3}{8} + \frac{2}{8} = \frac{5}{8}$$

## 异分母分数加减

**先通分，变成同分母分数，再加减。**

$$\frac{1}{3} + \frac{1}{4} = \frac{4}{12} + \frac{3}{12} = \frac{7}{12}$$

## 通分方法

1. 找出两个分母的最小公倍数
2. 将两个分数化成以最小公倍数为分母的分数

## 约分

把分数化成分子和分母都比较小但分数值不变的分数。

::: tip 示例
$$\frac{8}{12} = \frac{8 \div 4}{12 \div 4} = \frac{2}{3}$$
:::

**最简分数**：分子和分母只有公因数 1 的分数。
