# 分数乘除法

## 学习目标

1. 掌握分数乘法的计算方法
2. 掌握分数除法的计算方法
3. 能解决分数乘除法的实际问题

## 分数乘法

**分子相乘的积作分子，分母相乘的积作分母。能约分的先约分。**

$$\frac{3}{4} \times \frac{2}{5} = \frac{3 \times 2}{4 \times 5} = \frac{6}{20} = \frac{3}{10}$$

## 分数除法

**除以一个不为 0 的数，等于乘这个数的倒数。**

$$\frac{3}{4} \div \frac{2}{5} = \frac{3}{4} \times \frac{5}{2} = \frac{15}{8}$$

## 倒数

**乘积为 1 的两个数互为倒数。**

- $\frac{3}{4}$ 的倒数是 $\frac{4}{3}$
- 5 的倒数是 $\frac{1}{5}$
- 0 没有倒数

## 常见应用

**"求一个数的几分之几"** → 用乘法

**"已知一个数的几分之几是多少，求这个数"** → 用除法（或列方程）
