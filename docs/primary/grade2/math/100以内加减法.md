# 100 以内加减法

## 学习目标

1. 掌握 100 以内加减法的竖式计算方法
2. 理解进位加法和退位减法的算理
3. 能正确、熟练地计算

## 竖式加法（进位）

::: tip 示例：37 + 25
```
  3 7
+ 2 5
-----
  6 2
```
1. 个位：7 + 5 = 12，写 2 进 1
2. 十位：3 + 2 + 1 = 6
3. 结果：62
:::

## 竖式减法（退位）

::: tip 示例：52 - 38
```
  5 2
- 3 8
-----
  1 4
```
1. 个位：2 不够减 8，从十位借 1，12 - 8 = 4
2. 十位：5 借走 1 剩 4，4 - 3 = 1
3. 结果：14
:::

## 常见错误

**Q：孩子总是忘记进位或退位？**
A：建议在竖式上方标注进位 1 或退位点，形成习惯后会逐渐内化。
