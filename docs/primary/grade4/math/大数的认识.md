# 大数的认识

## 学习目标

1. 认识亿以内的数
2. 掌握数位顺序表
3. 会读写大数，比较大小
4. 会用"四舍五入"法求近似数

## 数位顺序表

| 亿级 | 万级 | 个级 |
|------|------|------|
| 亿位 | 千万位 | 百万位 | 十万位 | 万位 | 千位 | 百位 | 十位 | 个位 |

## 读数方法

1. 从高位读起，先读亿级，再读万级，最后读个级
2. 每级末尾的 0 不读，中间连续几个 0 只读一个

::: tip 示例
30050040 读作：三千零五万零四十
:::

## 四舍五入

- 省略的尾数最高位是 4 或小于 4 → 舍去
- 省略的尾数最高位是 5 或大于 5 → 向前一位进 1

## 常见问题

**Q：读数时 0 的处理容易出错？**
A：记住口诀"每级末尾 0 不读，中间 0 只读一个"。
