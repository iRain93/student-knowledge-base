# 万以内加减法

## 学习目标

1. 掌握万以内数的加减法竖式计算
2. 理解连续进位和连续退位
3. 会进行估算和验算

## 竖式加法

::: tip 示例：2456 + 1879
```
  2 4 5 6
+ 1 8 7 9
---------
  4 3 3 5
```
:::

## 竖式减法

::: tip 示例：4302 - 1856
```
  4 3 0 2
- 1 8 5 6
---------
  2 4 4 6
```
注意：连续退位时，某一位不够减，向前一位借 1 当 10。
:::

## 验算方法

- **加法验算**：交换加数位置再算一遍，或用和减去一个加数
- **减法验算**：用差 + 减数 = 被减数，或用被减数 - 差 = 减数
