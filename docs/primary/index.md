# 小学

小学阶段是数学知识体系构建的基础阶段，从具体运算逐步过渡到形式运算，为后续学习奠定扎实基础。

| 年级 | 核心内容 |
|------|---------|
| 一年级 | 20 以内加减法，认识图形和人民币 |
| 二年级 | 100 以内加减法，乘法口诀，长度单位 |
| 三年级 | 万以内加减法，多位数乘一位数，分数初步 |
| 四年级 | 大数的认识，三位数乘两位数，除法运算 |
| 五年级 | 小数乘除法，分数加减法，简易方程 |
| 六年级 | 分数乘除法，百分数，圆柱与圆锥 |

::: tip 学习建议
小学数学重在**理解算理**而非死记硬背，建议通过生活实例和教具操作帮助孩子建立数学概念。
:::
