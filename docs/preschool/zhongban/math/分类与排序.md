# 分类与排序

## 学习目标

1. 能按物体的颜色、大小、形状等特征进行分类
2. 能识别简单的排列规律
3. 能按规律继续排列

## 分类方法

| 分类标准 | 示例 |
|---------|------|
| 按颜色 | 红苹果 / 绿苹果 |
| 按大小 | 大球 / 小球 |
| 按形状 | 圆形积木 / 方形积木 |
| 按用途 | 吃的 / 玩的 |

## 教学活动建议

### 活动一：找不同
呈现一组物品，让孩子找出不属于同一类的那个。

### 活动二：串珠子
按规律串珠子，如红-蓝-红-蓝……

### 活动三：排队游戏
按高矮、性别等特征排队，发现规律。
