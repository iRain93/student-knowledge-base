# 高考复习专题

## 复习策略

高考数学复习一般分为三个阶段：

### 第一轮：系统复习（9月-次年2月）
- 逐个模块梳理知识点
- 夯实基础，不留死角
- 掌握基本题型和解题方法

### 第二轮：专题突破（3月-4月）
- 按题型分类训练
- 攻克重点难点
- 提升解题速度和准确率

### 第三轮：模拟冲刺（5月-6月）
- 全真模拟训练
- 查漏补缺
- 调整考试心态

## 六大核心模块

| 模块 | 高考分值 | 重点内容 |
|------|---|------|
| 函数与导数 | ~22 分 | 函数性质、导数应用、不等式 |
| 三角函数 | ~15 分 | 三角变换、解三角形 |
| 数列 | ~12 分 | 通项与求和、递推关系 |
| 解析几何 | ~22 分 | 圆锥曲线、直线与圆 |
| 立体几何 | ~17 分 | 空间向量、位置关系 |
| 概率统计 | ~12 分 | 排列组合、概率分布 |

## 常见压轴题类型

| 类型 | 解题策略 |
|------|---------|
| 导数综合题 | 构造函数、分类讨论 |
| 解析几何综合题 | 设而不求、韦达定理 |
| 数列综合题 | 数学归纳法、放缩法 |

## 考试技巧

1. **合理分配时间**：选择题 ~30 分钟，填空题 ~15 分钟，解答题 ~75 分钟
2. **先易后难**：确保基础题拿满分，再攻克难题
3. **规范答题**：书写工整，步骤完整，公式正确
4. **检查策略**：重点检查计算过程和单位
