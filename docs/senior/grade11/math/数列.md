# 数列

## 学习目标

1. 理解数列的概念
2. 掌握等差数列和等比数列的通项公式
3. 掌握数列求和的方法

## 数列的概念

按照一定顺序排列的一列数称为数列。

- **通项公式**：数列第 n 项 aₙ 与 n 之间的关系式
- **前 n 项和**：Sₙ = a₁ + a₂ + ... + aₙ

## 等差数列

| 项目 | 公式 |
|------|------|
| 通项公式 | aₙ = a₁ + (n - 1)d |
| 前 n 项和 | Sₙ = n(a₁ + aₙ)/2 = na₁ + n(n-1)d/2 |
| 等差中项 | 若 a, A, b 成等差，则 A = (a + b)/2 |

## 等比数列

| 项目 | 公式 |
|------|------|
| 通项公式 | aₙ = a₁·qⁿ⁻¹ |
| 前 n 项和 | Sₙ = a₁(1 - qⁿ)/(1 - q)（q ≠ 1） |
| 等比中项 | 若 a, G, b 成等比，则 G² = ab |

## 常见求和方法

| 方法 | 适用场景 |
|------|---------|
| 公式法 | 等差、等比数列 |
| 分组求和 | 可拆分为等差 + 等比 |
| 错位相减 | 等差 × 等比 |
| 裂项相消 | 分式数列 |
| 倒序相加 | 首尾配对 |
