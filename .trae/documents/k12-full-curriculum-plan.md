# K-12 全学科课程体系实施计划

## Context

当前知识库 15 个年级仅包含数学课程（每个年级 1-3 个知识点），需要按照中国国家课程标准，为每个年级补充完整的学科课程体系，包括所有学科的知识结构图和各知识点 md 文件。

## 实施概览

| 项目 | 数量 |
|------|------|
| 新增学科目录 | 76 个 |
| 新增知识结构.md | 76 个 |
| 新增知识点 md | ~245 个 |
| 更新年级 index.md | 15 个 |
| 更新首页 index.md | 1 个 |
| 更新 config.mjs 侧边栏 | 1 个（大幅扩展） |
| **总计** | **~342 个文件** |

## 各年级课程规划

### 幼儿园（小班/中班/大班）
每班 6 个学科：语言、数学、科学、艺术、健康、社会（每学科 3 个知识点）

### 小学 1-2 年级
8 个学科：语文、数学、英语、科学、道德与法治、音乐、美术、体育（每学科 3-4 个知识点）

### 小学 3-6 年级
9 个学科：语文、数学、英语、科学、道德与法治、音乐、美术、体育、信息技术（每学科 3-4 个知识点）

### 初一（7 年级）
10 个学科：语文、数学、英语、生物、历史、地理、道德与法治、音乐、美术、体育

### 初二（8 年级）
11 个学科：语文、数学、英语、物理、生物、历史、地理、道德与法治、音乐、美术、体育

### 初三（9 年级）
10 个学科：语文、数学、英语、物理、化学、历史、道德与法治、音乐、美术、体育

### 高一/高二/高三（10-12 年级）
13 个学科：语文、数学、英语、物理、化学、生物、历史、地理、政治、信息技术、音乐、美术、体育

## 侧边栏设计

采用可折叠分组结构，每个学科为一个组：

```js
'/primary/grade1/': [
  {
    text: '语文',
    collapsed: true,
    items: [
      { text: '知识结构', link: '/primary/grade1/chinese/知识结构' },
      { text: '汉语拼音', link: '/primary/grade1/chinese/汉语拼音' },
      // ...
    ]
  },
  // ... 数学、英语、科学等
]
```

## 文件内容模板

每个知识点 md 文件包含：`# 标题` > `## 学习目标` > `## 核心内容`（含 ::: tip 和表格） > `## 常见问题`

每个 知识结构.md 包含：Mermaid 知识图谱、知识点列表表格、学习目标

## 实施步骤

1. 按年级批量创建学科目录和文件（幼儿园 → 小学 → 初中 → 高中）
2. 更新 config.mjs 侧边栏配置
3. 更新各年级 index.md 添加所有学科
4. 更新首页
5. 构建验证

## 验证方式

- `npx vitepress build` 无报错
- 每个年级侧边栏正常显示所有学科
- 所有知识点页面可正常访问
- Mermaid 图表正常渲染
- 面包屑导航正确显示学科名
